# Работа с ИИ-агентом `cursor на базе нескольких моделей в режиме auto: Composer 2 Fast, GPT-5.3 Codex, Sonnet 4.6, Opus 4.6`

Применение агента происходило после написания кода с целью определения альтернативных вариантов решения.
Реализация без агента имела императивный подход и как следствие больший объём кодовой базы.
Применение агента для частичной генерации кода и рефакторинга позволило сделать код более лаконичным
благодаря использованию сторонних библиотек.

Примеры промптов:

1. В компоненте @src/pages/LoginPage/LoginPage.tsx использовать компонент формы mui FormControl
2. Заменить состояния useState хуком из react-hook-form в @src/pages/LoginPage/LoginPage.tsx,
   сохранив работоспособность - задача упростить императивную реализацию value и onChange.
3. Создать хук для реализации логики взаимодействия с api, используя инструменты @tanstack/react-query.
4. Корректировка стилей. Поиск default отступа по границам верхнеуровнего конмонента и его удаление.
5. Корректировка логики авторизации. Добавление ошибок на русском языке и разбиение по полям login и password.
6. Обособить логику авторизации в хук useAuthStore.
